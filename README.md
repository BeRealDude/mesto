# Проект: Место. Поделись тем, что увидел.

**В этом проекте можно:**
* Редактировать информацию профиля.
* Добавлять свои фотографии и удалять их.
* Ставить на фотографии лайки.

### При создании использовались
* Flex
* БЭМ
* Grid Layout
* Медиазапросы
* Адаптивная вёрстка
* Java Script
